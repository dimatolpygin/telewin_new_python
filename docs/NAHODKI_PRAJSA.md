# Сводная карта находок (прочёс прайса 5 субагентами Haiku, 2026-07-21)

Источник наполнения словарей/тестов для апгрейда поиска. 11 864 позиции, 11 групп.

## 1. АБСТЕЙН-ЛОВУШКИ (риск №1) — реальные имена, всплывающие на чужой запрос
Ключ: часть — РЕАЛЬНЫЙ ассортимент (мыло, белизна, губки, репелленты — магазин продаёт).
Ловушка срабатывает на ЧУЖОЙ квалификатор: «шампунь ДЛЯ ВОЛОС», «масло ДЛЯ ВОЛОС».
Различать по контексту, не глушить слово целиком.
- Масла: «Масло пихтовое 50/100мл», «Масло бытовое 100мл», «Масло 2Т 1л», «Масло силиконовое 15мл» → чужое: масло для волос/косметическое/пищевое
- «Шампунь д/моек Carver 1л» → чужое: шампунь для волос
- «Мыло туалетное/хозяйственное» → это ассортимент, но пересекается с гигиеной
- «Мочалка банная», «Губка д/посуды», «Бумага туал.» → чужое: мочалка для тела, губка косметическая
- «Ошейник» ×15 (брезент, для животных) → чужое: ошейник-украшение
- «Белизна», «Порошок стирал.», «Гель д/стирки» → ассортимент дома, но звучит бытово
- Инструмент: «Щётка электр. графитовая» (для двигателей!) → чужое: зубная щётка электрическая
- Инструмент-омонимы: «Уровень», «Линейка», «Рулетка», «Панель», «Шина», «Колодка» (эл.)
- Электро: «Розетка» (электро vs сантех), «Фильтр» (сетевой vs воздушный)

Сантех резьба: г/г=гайка-гайка, г/ш=гайка-штуцер, ц/г=цанга-гайка, в/н=внутр/наружн
Электро: с/з=с заземл., б/з=без, с/у=скрытая установка, 1кл/2кл=клавиши, п=полюс(автомат), К=Кельвин

## 4. АТРИБУТЫ (парсер) — где движок ошибается
- **Дюймы 1/2 vs 1½** — КРИТИЧНО: это 15мм vs 40мм, стеммер slovar их слепляет. Также 3/4, 1", 1¼
- **Цоколь ламп** — главный атрибут электрики: E27(большой)/E14(малый)/G4/G9/GU10/GX53(точечный)/T5,T8(трубка)
- **Цветовая температура**: тёплый→2700К, нейтральный→4000К, холодный→6500К
- **Размеры крепежа**: 3.5≠4.0≠4.2≠4.8 (разные саморезы), М8≠М10≠М12 — брать ЗАПРОШЕННОЕ число, не первое
- **Уголок 40*40*80**: 3-е число = высота ребра, НЕ толщина (толщина ~2мм отдельно)
- **Материал = жёсткий фильтр**: ПП/ПЭ/м/п/сталь/латунь невзаимозаменяемы; дерево/бетон/металл (сверло)
- Сечение кабеля 3х2.5, ток автомата 16А, мощность Вт, IP-защита, зернистость Р120

## 5. БРЕНДЫ
- Народных Makita/DeWalt/Bosch/Hilti/Stanley/Зубр в прайсе НЕТ (Bosch — 1 диск). На них честный ответ:
  «такого бренда нет», НЕ молчание и НЕ подмена.
- Реальные бренды достойные канала: БИБЕР, Вихрь, MATRIX, СИБРТЕХ, FIT, SPARTA, ПРАКТИКА (инстр.);
  Valtec, JIF, LIKE (сантех); Rexant, EKF, ЭРА, Navigator, IN HOME, Feron, IEK (электро); Тех-Креп, Rennbohr (крепёж)
