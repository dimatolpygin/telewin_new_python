| Режим | Каналы | top-1 |

Подгруппа в одиночку слаба (recall-подпорка), но **не роняет** top-1 при добавлении к словарю —
поэтому включена в проде по умолчанию (даёт recall на родовые запросы).

## Новые каналы — что умеет поиск сверх TS

- **Штрихкод (EAN):** запрос из 8–13 цифр → точный товар (в новом прайсе EAN 100%).
- **Подгруппа Форы (recall):** родовой запрос «нужен крепёж» → товары подгруппы «Крепёж»
  (ИИ дальше уточняет). ЛКМ-запросы («масляная»/«эмульсионная краска») → товары ЛКМ.
- **Производитель (fallback):** «что есть от бибера» → товары БИБЕР (573 позиции),
  канал «производитель». Срабатывает, когда производитель полностью назван и словарь молчит.

## Остаточные 2 «провала» top-1 — корректное поведение

- **«крепеж»** — родовой запрос: поиск отдаёт товары крепежа (recall), точное семейство
  выбирает уже диалог с ИИ. Не ошибка поиска.
- **«что есть от бибера»** — запрос по производителю: отдаём товары БИБЕР (top-3 содержит
  позиции со словом БИБЕР в имени). Regex золотого набора именной, поэтому top-1 не совпал.

## Известное ограничение (по CLAUDE.md — вычитка вручную)

Резкое разведение **масляная (эмаль ПФ)** vs **эмульсионная (В/Д)** словарём достигнуто
частично: токен «краска» перетягивает семейство «Краска», поэтому оба запроса отдают краски
из подгруппы ЛКМ, но не строго ПФ vs В/Д. Полное разведение — задача доменной вычитки отдела ЛКМ,
не автоматической правки.
