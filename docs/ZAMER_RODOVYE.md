# Замер этапа 16 — Категорийный канал для родовых + уточнение (финал milestone 2)

Дата: 2026-07-21. Последний этап milestone 2. Инвариант — золото ≥95% top-1 без регресса.

## Что было (проблема)

Родовые запросы ДЕЙСТВИЕМ/НАЗНАЧЕНИЕМ не резолвились, когда стем глагола ≠ стему семейства:
«чем красить» (стем `крас`) не находил семейство Краска (стем `краск`) → **пусто**; так же
«чем прикрутить», «чем приклеить». Родовые же с десятками несовместимых вариантов («нужен болт»,
«краска», «дюбель») агент мог отдавать наугад одной позицией.

## Что сделано

1. **Канал родовых действий (`data/rodovye.json` + `search.py`).** Стем глагола → список
   семейств-кандидатов (крас/покрас→Краска/Эмаль/Валик/Кисть; прикрут→Шуруп/Дюбель/Болт;
   прикле/скле→Клей; грунтова→Грунт/Грунтовка; штукатур/шпаклева→Шпатлевка). Мягкая инъекция
   (вес 1.0): семейства добавляются в кандидаты и конкурируют в скоринге, НЕ короткозамыкают.
   Глаголы сверлить/пилить сюда НЕ внесены — уже резолвятся словарём (стем=семейству).
2. **Безопасное уточнение (`ai/agent.py`).** Правило в промпте: на родовом с десятками
   несовместимых вариантов (болт/краска/дюбель/кабель/лампа) сначала уточнить ключевой параметр
   (диаметр М.., тип краски, сечение, цоколь), а не выдавать наугад. Показ без уточнения — только
   если вариантов мало.

## Проверка критериев

| Запрос | Было | Стало |
|---|---|---|
| `чем сверлить` | Сверло ✅ | Сверло (в тему top-3) ✅ |
| `от тараканов` | Ловушка/Дуст ✅ | средства от вредителей (top-3) ✅ |
| `нужен крепёж` | Крепеж ✅ | Крепеж (top-3) ✅ |
| `чем красить` | **пусто** | **Краска/Эмаль** ✅ |
| `чем прикрутить` | **пусто** | **Шуруп** ✅ |
| `чем приклеить` | **пусто** | **Клей** ✅ |

**Живой смоук агента** (`run_agent`, haiku):
- «нужен болт» → уточняет диаметр/длину (М6/М8/М10) и назначение, случайный болт НЕ выдаёт ✅.
- «чем красить стену водоэмульсионкой» → предлагает выбор (кисти/валики/грунтовка/краска) с уточнением ✅.
- «от тараканов» → поиск «средство от тараканов», выдача гель/ловушка/дуст ✅.
