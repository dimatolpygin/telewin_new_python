## Что добавлено в парсер (`attributes.py` / `search.py` / `sleng_razmerov.json`)

1. **Цоколь ламп** — было только Е14/27/40, стало +G-цоколи: GU10, GU5.3, GX53, GX70,
   G9, G4, G23, G13, T8, T5, R7S. Нормировка в единый канон (Е→E, без пробелов,
   запятая→точка). Читается из имени (`razobrat`, `re.I`) и из запроса
   (`atributy_zaprosa` по `q.lower()` — **не** по `norm`: гомоглиф `x→х` ломал «gx53»).
2. **Цветовая температура** — новый атрибут `цв_температура`: «2700К»/«4000K»/«6500 К»
   из имени (только с якорем К); из запроса — слова тёплый→2700, нейтральный/естественный→4000,
   холодный/дневной→6500 (`sleng['цв_температура']`).
3. **Ток автомата** — новый атрибут `ток_а`: «16А»/«25 А»/«40А» (цифра перед А, не путать
   с «ВА 47-29»). Чинит «автомат 16а/40а»→верный номинал (было 25А).
4. **Дюймы 1/2 vs полтора дюйма** — прайс пишет полтора как `1 1/2"` или `1" 1/2 (40)` (НЕ
   символом ½). Целое ПЕРЕД дробью → отдельный размер «1 1/2» (40мм) ≠ «1/2» (15мм). Сленг
   «полтора дюйма»/«полторашка»→«1 1/2». Закрывает ловушку CLAUDE.md (стеммер слеплял 15↔40мм).
5. **Гард артикула по температуре** — 2700/4000/6500 — 4 цифры и раньше короткозамыкали на
   артикул-канал («лампа тёплый свет 2700»→товар с артикулом 2700). При лампа-контексте
   (`ламп|свет|цокол|тепл|холодн|нейтральн`) эти числа артикулом не считаются.
6. **Падежи гипсокартон** — `гипсокартону/гипсокартоном/гипсокартоне→ШСГД` в `слово_в_имени`
   (форма из размерного набора: «саморез по гипсокартону 3.5» была →ШСГМ, стала →ШСГД).

Скоринг уже обрабатывает любой атрибут `qa` автоматически (совпадение +4 / конфликт −1.5 +
голое число +2), поэтому новые ключи (цоколь/цв_температура/ток_а) заработали без правок скоринга.

## Критерии приёмки (probe)

- Все 9 лампа-кейсов размерного набора OK: E27/E14/GU10/G9/GX53/G4 + тёплый 2700 /
  нейтральный 4000 / холодный 6500.
- «кран шаровый полтора дюйма 40» → Кран шар. 1" 1/2 (40); «кран водоразб. 1/2» → 1/2 (не 1½).
- «автомат 16а»→16А, «автомат 40а»→40А (ампераж). «болт м12»→М12 (не М10).

## Остаточные промахи размерной точности (3/32, вне цели)

- «болт м10 100мм» → Болт **Мебельный** M10*100 — размер (M10*100) верный, тип «мебельный»
  вместо обычного; `ждём ^Болт M10` строг к типу.
- «саморез 4.8 длинный» → Шуруп сантех. 10*100 — одиночный диаметр 4.8 без «*» слабо
  фильтрует против «длинный»→100мм.
- «саморез по металлу 4.2» → ШСММ 4,2*50 (ждём ШСГМ) — оба металл-самореза; `ждём` спорен
  (ШСММ = металл-металл семантически ближе «по металлу», чем ШСГМ = гипсокартон-металл).

Не чинились: цель 91% уже взята, а правки под них рискуют регрессом. Кандидаты на этап 16
(уточнение) / ревизию `ждём` набора.
