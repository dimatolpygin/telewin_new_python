# Замер: лексика vs гибрид (этап 10 — pgvector + RRF)

**Дата**: 2026-07-20. **Наборы**: золотой (40 товарных + 2 мусорных), слепой `test_nabor`
(279 товарных + 23 мусорных, дорощен со 155 в сквозной задаче v2). Метрика — попал@1 / попал@3
(регэксп `ждём` по имени товара), плюс абстейн на мусоре (успех = ничего не вернули).

## Что сравниваем

- **лексика** — детерминированное ядро этапов 1–9 (словарь домена + атрибуты + фаззи-канал
  опечаток + канал подгруппы). `Poisk.iskat`.
- **гибрид** — та же лексика, слитая с **векторным каналом** (эмбеддинг запроса → kNN по
  pgvector, `halfvec(3072)`, косинус) через **RRF** (Reciprocal Rank Fusion). `Gibrid.iskat`.

Эмбеддинги: OpenRouter `google/gemini-embedding-001` (3072), текст — обогащённое имя
(раскрытие сокращений «по дер.»→«по дереву» + таксономия семейство/подгруппа/группа).

## Параметры слияния (подобраны свипом)

`RRF: score(d) = Σ_канал w / (K + rank)`, `K=60`, `W_LEX=1.0`, `W_VEC=0.3`,
порог чисто-векторного ответа `0.76`.

- **W_VEC=0.3** — при большем весе вектор перебивает точный лексический матч и золотой
  набор регрессирует 38→37 (напр. «саморез черный 45» → вектор тянул «ШУЦ 4,5*45» вместо
  верного «ШСГД …*45»). При 0.3 золото не страдает, а слепой всё равно растёт.
- **порог 0.76** — восстанавливает абстейн на мусоре (без порога kNN всегда что-то возвращает).
  На товарный top-1 порог не влияет: ни один товарный запрос не выигрывается чисто-векторным
  путём — вектор помогает через ре-ранжирование внутри лексических кандидатов.
- Жёсткие/прямые каналы (штрихкод, артикул, производитель) короткозамыкают — вектор в них
  не вмешивается.

## Результаты

### ЗОЛОТОЙ (40 товарных, 2 мусорных)
| система | top-1 | top-3 | абстейн мусор |
|---|---|---|---|
| лексика | 38/40 = 95% | 39/40 = 98% | 2/2 |
| **гибрид** | **38/40 = 95%** | **39/40 = 98%** | 2/2 |
| Δ | +0 | +0 | +0 |

Регрессии нет — жёсткие каналы и точные размеры не деградируют.

## Что вытащил вектор (примеры, лексика промахивалась)

- «пассатижи 160 мм» → Плоскогубцы 160 мм (пассатижи = плоскогубцы, слова в прайсе нет)
- «болгарка щётка по металлу» → щётка для УШМ / металлическая (болгарка = УШМ)
- «гофротруба кабельная d20» → Гофротруба с зондом d=20
- «бита филипс 25мм магнит» → Бита Ph1x25 (филипс = PH)
- «морилка для дерева венге» → Морилка …, д/дерева Венге
- «пена монтажная кудо 1000ml» → Пена монт. проф. "KUDO" … 1000мл
- «смеситель для кухни с длинным изливом» → Смеситель кух. … с высоким изливом
- «автоматический выключатель 3 полюса 32» → Автом. выкл. … АД-32
- «метла синтетическая без черенка» → Метла … на берёз. черенке

Единственный «слом» — «отвертка крестовая 200мм» → «Отвертка PH2*200 мм» (фактически верная
крестовая отвёртка 200 мм; промах лишь регэкспа-ожидания, не смысла).


## Вывод

Гибрид строго выше чистой лексики на слепом наборе (top-1 +11, top-3 +4), золотой набор
без регрессии (95%), абстейн на мусоре сохранён. Открытые синонимы и композиция, недоступные
словарю, ловятся вектором. Критерии приёмки этапа 10 выполнены.
